# Lab 03: IPv6 Addressing

**Module:** M05 | **Course:** IT128 | **Date:** September 28, 2025

## Tools Used
- pfSense
- Windows ipconfig commands
- NDG NetLab (pfSense + WinOS)

## What I Did
Configured a Windows computer to use IPv6 addressing through both
DHCPv6 and static IPv6 configuration. Used ipconfig/all,
ipconfig/release6, ipconfig/renew6, and ping commands to confirm
the client successfully received addresses via DHCPv6 and could
release and renew IPv6 leases.

## Key Concepts
- IPv6 address structure and notation
- DHCPv6 dynamic address assignment
- Static IPv6 configuration
- ipconfig/release6 and ipconfig/renew6 commands
- IPv6 vs IPv4 differences

## Why This Matters
IPv6 is the present and future of networking. Understanding how to
configure and troubleshoot IPv6 addressing is required for Network+
and is increasingly expected in enterprise networking roles as IPv4
address space continues to be exhausted.
