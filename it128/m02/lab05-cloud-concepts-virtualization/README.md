# Lab 05: Cloud Concepts and Virtualization

**Module:** M02 | **Course:** IT128 | **Date:** September 7, 2025

## Tools Used
- Microsoft Hyper-V
- Xubuntu Linux
- NDG NetLab (Hyper-V on WinOS)

## What I Did
Deployed a virtual machine using Hyper-V in the NDG NetLab environment.
Created and configured a new VM, installed Xubuntu Linux, verified
successful boot, reviewed and modified hardware settings including
memory allocation, practiced VM lifecycle functions including power on
and shutdown, and created and reverted production checkpoints.

## Key Concepts
- Virtual machine creation and configuration in Hyper-V
- VM hardware settings — memory allocation, add hardware
- VM lifecycle management — power on, shutdown, turn off
- Production checkpoints for rollback and recovery
- Hyper-V Manager interface and VM naming

## Why This Matters
Virtualization is the backbone of modern IT infrastructure and cloud
computing. Understanding how to deploy and manage VMs in Hyper-V
maps directly to Azure and cloud administration skills. Checkpoints
mirror real-world change management practices where rollback plans
are required before any update.
