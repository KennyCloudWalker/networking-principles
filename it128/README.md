# IT128 — Network+ (Networking Principles)

**Institution:** Coastline Community College
**Term:** Fall 2025
**Instructor:** Dr. Barrios
**Certification Alignment:** CompTIA Network+
**Lab Environment:** NDG NetLab, GNS3, VyOS, pfSense

---

## Course Overview

Hands-on networking labs covering the CompTIA Network+ exam domains
including OSI model, IP addressing, routing, switching, NAT, VLANs,
ACLs, network availability, AAA authentication, and troubleshooting.
