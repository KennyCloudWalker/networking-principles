# Lab 12: AAA Authentication

**Module:** M13 | **Course:** IT128 | **Date:** November 23, 2025

## Tools Used
- pfSense
- NDG NetLab (WinOS + pfSense VM)

## What I Did
Logged into pfSense web GUI, navigated to the AAA configuration
section, restored a provided configuration backup, and configured
Authentication, Authorization, and Accounting services. Verified
the AAA setup was functioning correctly through the pfSense dashboard.

## Key Concepts
- AAA framework — Authentication, Authorization, Accounting
- pfSense AAA configuration
- Configuration backup and restore in pfSense
- Centralized authentication management
- Role-based access control principles

## Why This Matters
AAA is the framework that controls who can access a network, what they
can do, and tracks what they did. It is the foundation of network
access control in every enterprise environment. Understanding AAA is
required for Network+ and Security+ and is directly applicable to
network administrator and security operations roles.
