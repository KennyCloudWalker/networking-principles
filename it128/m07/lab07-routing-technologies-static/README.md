# Lab 07: Routing Technologies — Static Routing

**Module:** M07 | **Course:** IT128 | **Date:** October 12, 2025

## Tools Used
- VyOS Routers
- GNS3
- NDG NetLab

## What I Did
Configured routing technologies using VyOS routers in a GNS3 simulated
environment. Established communication between two separate LANs by
configuring IPv4 and IPv6 addressing, static routes, and default routes.
Verified proper routing table population and confirmed successful
connectivity between all hosts across both networks.

## Key Concepts
- Static route configuration on VyOS
- Default route configuration
- IPv4 and IPv6 routing
- Routing table verification with show commands
- Inter-LAN communication through a router

## Why This Matters
Routing is how networks communicate with each other. Configuring static
routes on a real router platform like VyOS in GNS3 mirrors what network
engineers do in enterprise environments. Understanding routing tables
and how packets find their path is core Network+ knowledge and directly
applicable to network administrator and engineer roles.
