# Lab 08: Routing Technologies — RIP, OSPF, and BGP

**Module:** M09 | **Course:** IT128 | **Date:** October 27, 2025

## Tools Used
- VyOS Routers
- GNS3
- NDG NetLab

## What I Did
Configured and verified RIP, OSPF, and BGP dynamic routing protocols
using VyOS routers in GNS3. Practiced setting up IPv4 and IPv6 routing
for each protocol, verified connectivity with show commands and pings,
and observed how each protocol exchanges routes differently and adapts
automatically to topology changes.

## Key Concepts
- RIP — distance vector routing protocol
- OSPF — link state routing protocol
- BGP — path vector routing used on the internet
- Dynamic vs static routing comparison
- show ip route and show ip ospf neighbor verification
- IPv4 and IPv6 dynamic routing configuration

## Why This Matters
Dynamic routing protocols are how large networks automatically manage
routing without manual configuration. Understanding RIP, OSPF, and BGP
is required for Network+ and is foundational knowledge for network
administrator, network engineer, and NOC analyst roles. BGP is the
protocol that runs the entire internet.
