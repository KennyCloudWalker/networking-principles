# Lab 01: OSI Model and Encapsulation Concepts

**Module:** M01 | **Course:** IT128 | **Date:** August 28, 2025

## Tools Used
- Wireshark
- NDG NetLab (Windows 10 VM)

## What I Did
Used Wireshark to capture and analyze live network traffic, identifying
how data is encapsulated through each layer of the OSI model. Observed
ARP at Layer 2, TCP three-way handshake at Layer 4, DNS/UDP at Layer 4,
HTTP GET requests at Layer 7, and ICMP ping requests at Layer 3.

## Key Concepts
- OSI model layers and their functions
- Ethernet frame structure (Layer 2 — Data Link)
- IP header analysis (Layer 3 — Network)
- TCP segment analysis (Layer 4 — Transport)
- UDP/DNS connectionless transport (Layer 4)
- HTTP application traffic (Layer 7)
- ICMP network diagnostics (Layer 3)

## Why This Matters
Wireshark is the industry-standard packet analyzer used by network
engineers and security analysts daily. Understanding how data moves
through each OSI layer is foundational knowledge for every networking
and security certification and role.
