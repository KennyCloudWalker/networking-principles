# Lab 10: Controlling Traffic and Creating ACLs

**Module:** M11 | **Course:** IT128 | **Date:** November 9, 2025

## Tools Used
- VyOS Router (R1)
- EXOS Switch (R2)
- GNS3
- NDG NetLab

## What I Did
Configured firewall filtering and Access Control Lists in a simulated
enterprise network using GNS3. Applied firewall rules on VyOS to block
ICMP traffic, then created and applied both static and dynamic ACLs on
EXOS to regulate IPv4 host traffic between subnets. Verified all
traffic control configurations through connectivity testing.

## Key Concepts
- Firewall rule configuration on VyOS
- Static ACLs for fixed traffic filtering
- Dynamic ACLs for flexible traffic control
- ICMP traffic blocking and verification
- ACL application to network interfaces
- Traffic flow verification between subnets

## Why This Matters
ACLs and firewall rules are the primary tools for controlling what
traffic is allowed on a network. Every enterprise network uses ACLs
to enforce security policy. This is core Network+ and Security+
content and a daily task for network administrators and security
engineers.
