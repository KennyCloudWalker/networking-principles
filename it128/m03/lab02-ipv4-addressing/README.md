# Lab 02: IPv4 Addressing

**Module:** M03 | **Course:** IT128 | **Date:** September 12, 2025

## Tools Used
- Windows Control Panel
- Linux Terminal (MintOS)
- NDG NetLab

## What I Did
Configured and verified IPv4 addresses on both Windows and Linux
systems in the NDG NetLab environment. Set static and dynamic IP
addresses, identified default gateways and subnet masks, and verified
connectivity using command line utilities.

## Key Concepts
- Static vs dynamic IPv4 addressing
- Subnet masks and default gateways
- ipconfig (Windows) and ifconfig/ip (Linux)
- Verifying connectivity with ping
- IPv4 address classes and CIDR notation

## Why This Matters
IPv4 addressing is the foundation of every network. Configuring and
troubleshooting IP addresses on both Windows and Linux systems is a
daily task for network administrators, help desk technicians, and
security analysts. This is core Network+ exam content.
