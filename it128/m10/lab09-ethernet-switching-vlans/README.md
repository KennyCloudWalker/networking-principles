# Lab 09: Ethernet Switching — VLANs, Trunking, and Port Security

**Module:** M10 | **Course:** IT128 | **Date:** November 2, 2025

## Tools Used
- EXOS Switch (ExtremeXOS)
- NDG NetLab

## What I Did
Configured and validated Layer 2 switching technologies on an EXOS
switch. Explored how MAC addresses are learned and how switching
decisions are made. Configured port security, VLANs, trunking, remote
management access via Telnet and SSH, and link aggregation. Verified
all configurations using show commands.

## Key Concepts
- MAC address learning and switching decisions
- VLAN creation and port assignment
- Trunk port configuration for inter-switch VLAN traffic
- Port security to restrict unauthorized devices
- Telnet and SSH remote management access
- Link aggregation for redundancy and bandwidth

## Why This Matters
VLANs and switching are the foundation of every enterprise network.
Segmenting traffic with VLANs improves security and performance.
Port security prevents unauthorized devices from connecting. These
are core Network+ topics and daily tasks for network administrators
and network engineers.
